# Air Quality Prediction using Machine Learning

## 📌 Project Overview
This project predicts Air Quality Index (AQI) using:
- ARIMA (Time Series Forecasting)
- LSTM (Deep Learning Model)

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / PyTorch

## 🎯 Goal
To compare traditional time-series forecasting with deep learning models.
